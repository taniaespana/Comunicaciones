# ✦ Comunicado: Guía de Canales de Comunicación - CAMDP

Página estática con la guía de canales de comunicación recomendados para el equipo CAMDP.

## 📚 Contenido

- **Resumen visual** de los 4 canales: Teams, Correo, Ticket/Jira, Reuniones
- **Detalle por canal**: cuándo sí y cuándo no usarlos
- **Flujo de decisión**: 4 pasos para elegir el canal correcto
- **Escenarios comunes**: 10 situaciones reales con canal sugerido
- **Reglas de oro**: principios de comunicación efectiva

## 🚀 Ver la página

Simplemente abre `index.html` en tu navegador o accede al link de GitHub Pages.

## 🛠️ Tecnologías

- HTML5
- Tailwind CSS (CDN)
- Font Awesome (CDN)
- Colores oficiales de Walmart
